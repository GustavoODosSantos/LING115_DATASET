# LING115_DATASET
# LING 115 Final Project Dataset

This repository contains the dataset used for a corpus-based analysis of modal verbs in English and Portuguese.

## Data
- Source: NOW Corpus (English-Corpora.org, CorpusdoPortugues.org)
- 500 KWIC examples per modal verb:
  - must
  - should
  - deve
  - precisa

## Method
The dataset is used for manual annotation and frequency analysis of meaning categories:
- obligation
- recommendation
- probability
- other

## Tools
- Python (pandas)
- Excel
